\# Cidade Viva



Toda cidade possui informações importantes que são difíceis de encontrar: locais acessíveis, pontos de descarte de resíduos, iniciativas comunitárias e serviços úteis. O Cidade Viva nasceu para reunir essas informações em uma plataforma colaborativa, confiável e acessível, colocando as pessoas no centro das decisões.



\## Status do projeto



🚧 Em desenvolvimento inicial.



\## Stack



\- Backend: NestJS + TypeScript + PostgreSQL + Prisma

\- Frontend: React + TypeScript + Vite
